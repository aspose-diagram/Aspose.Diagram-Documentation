## **مقدمة**
### **IntelliJ IDEA**
**IntelliJ IDEA** هي بيئة تطوير متكاملة Java (IDE) لمطوري البرمجيات. تم تطويره بواسطة JetBrains. تم إنشاؤه في الأصل for Java ، وهو يدعم حاليًا مجموعة متنوعة من اللغات ، بما في ذلك JavaScript و Groovy و HTML و CSS و RSS و R و Haskell و PHP و Ruby و Python و Scala و Clojure و Kotlin.

 في عام 2010**عالم المعلومات** تقرير ، حصل IntelliJ على أعلى درجة في مركز الاختبار من بين أفضل أربع أدوات برمجة Java: Eclipse و IntelliJ IDEA و NetBeans و Oracle JDeveloper.
### **Aspose.Diagram for Java**
[Aspose.Diagram](https://products.aspose.com/diagram/java/)for Java هو API غير رسومية وجيد التنظيم يسمح للمطورين بتحميل diagram موجود ، وإنشاء diagram من البداية ، وحفظ diagram بأي تنسيق ملف مدعوم ، والتعامل مع رسومات Microsoft Visio داخل تطبيق Java دون استخدام Visio. أداء أفضل وأسهل في التعامل مع الرسوم البيانية وتحويل الملفات من Microsoft Office الأتمتة.
### **Aspose.Diagram Java لـ IntelliJ IDEA - Maven**
-  Aspose.Diagram Java لـ IntelliJ IDEA (Maven) هو**البرنامج المساعد IntelliJ IDEA** الذي يتيح لك إنشاء مشاريع maven (أي يحتوي المشروع الذي تم إنشاؤه على Aspose.Diagram for Java API maven التبعية) وتنزيل أمثلة التعليمات البرمجية المصدر لـ API للاستخدام[Aspose.Diagram for Java API](https://products.aspose.com/diagram/java/).
-  هذا البرنامج المساعد يساعدك على العمل مع[Aspose.Diagram for Java API](https://products.aspose.com/diagram/java/) داخل فكرة IntelliJ IDEA المفضلة لديك.

## **متطلبات النظام والأنظمة الأساسية المدعومة**
### **متطلبات النظام**
- Microsoft Windows 10/8/7 / Vista / 2003 / XP (بما في ذلك 64 بت) / Linux GNOME أو KDE لسطح المكتب / Mac OS X 10.5 أو أعلى
- 1 غيغابايت كحد أدنى من ذاكرة الوصول العشوائي و 2 غيغابايت من ذاكرة الوصول العشوائي
- مساحة 300 ميجا بايت على القرص الصلب + 1 جيجا على الأقل للتخزين المؤقت
- الحد الأدنى لدقة الشاشة 1024 × 768
- يوصى بـ 1.8 دينار أردني أو أعلى
- اتصال بالإنترنت (يوصى بـ 2 ميغا بايت أو أسرع)

## **جارى التحميل**
### **تنزيل IntelliJ IDEA**
 ستحتاج أولاً إلى تثبيت IntelliJ IDEA CE / UE قبل تنزيل ملف**Aspose.Diagram Java لـ IntelliJ IDEA (Maven)** توصيل في.

لتحميل IntellliJ IDEA

1.  اذهب إلى<https://www.jetbrains.com/idea/download/>
1. حدد نظام التشغيل الخاص بك قيد التشغيل
1. انقر فوق تنزيل UE أو CE.
### **تنزيل Aspose.Diagram Java للمكون الإضافي IntelliJ IDEA (Maven)**
-  يمكن تنزيل البرنامج المساعد مباشرة من ملف[عنوان URL لاستضافة البرنامج المساعد JetBrains](https://goo.gl/JjSReR) - في مستودع البرنامج المساعد IntelliJ IDEA
والإصدارات اللاحقة يمكن تثبيتها باستخدام خيار IDE:
 - اذهب إلى**ملف** > **إعدادات**...>**الإضافات**
 - يختار "**تثبيت البرنامج المساعد من القرص "**
-  أو يمكن تثبيته مباشرة (بدون تنزيل يدوي من عنوان URL لاستضافة المكون الإضافي JetBrains) باستخدام خيارات IDE:
 - اذهب إلى**ملف** > **إعدادات**...>**الإضافات**
 - يختار**"تصفح المستودعات"**
 - في مربع البحث ، اكتب: "Aspose"
 - يختار "**Aspose.Diagram Java لـ IntelliJ IDEA Maven**"البرنامج المساعد للتثبيت.
