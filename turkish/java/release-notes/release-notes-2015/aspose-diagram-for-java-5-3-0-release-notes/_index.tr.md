(DIAGRAMJAVA-50155) - VSDX'den SVG'e dönüştürme, köprülerin hatalı oluşturulması

(DIAGRAMJAVA-50156) - Java ZIP Paketinden Örnek Panoyu Kaldır

(DIAGRAMJAVA-50159) - VST - VDX dönüştürme, makro modülleri korunmaz
