 Entwickler können ActiveX-Steuerelemente direkt zu Microsoft Visio-Zeichnungen hinzufügen, um Visio diagram interaktiv zu gestalten[Aspose.Diagram for .NET API](https://products.aspose.com/diagram/net/).

{{% /alert %}}
