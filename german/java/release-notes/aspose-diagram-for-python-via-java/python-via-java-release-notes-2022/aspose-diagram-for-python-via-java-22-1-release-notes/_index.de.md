### **Fügt setSaveAsSingleFile in HTMLSaveOptions hinzu**
- Gibt an, ob die HTML-Datei als einzelne Datei gespeichert werden soll.

{{< highlight "java" >}}

    HTMLSaveOptions o = new HTMLSaveOptions();    
    o.setSaveAsSingleFile(true);
