## **Installation**
### **Windows:**
You can easily use Aspose.Diagram for Python via .NET from [pypi](https://pypi.org/project/aspose-diagram-python/) with the following command.
{{< highlight NET >}}

 $ pip install aspose-diagram-python

{{< /highlight >}}

### **Linux:**
You can easily use Aspose.Diagram for Python via .NET from [pypi](https://pypi.org/project/aspose-diagram-python/) with the following command.
