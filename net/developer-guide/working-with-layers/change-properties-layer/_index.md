## **Change Layer Properties in Visio**
[Aspose.Diagram for .NET](https://products.aspose.com/diagram/net/) allows to change layer's properties in Microsoft Office Visio diagram.Each shape can belong to multiple layers so developers can change layer's properties to suit end user needs. The [PageSheet](https://apireference.aspose.com/diagram/net/aspose.diagram/pagesheet) class object offers Layers which allows to add and remove layer objects in Visio drawing.
