## **添加了一个新的 Page 类构造函数以传递最大页面 ID**
开发人员可以轻松计算最大页面 ID 并递增它。新的 Page (int ID) 构造函数允许您传递页面 ID。您可以查看有关此功能的示例主题：[Insert a New Blank Page into a Visio Drawing]
