Aspose.Diagram for .NET 是一个编程库，允许软件开发人员从他们自己的应用程序中操作和转换 visio 文件。 Aspose.Diagram for .NET supports the popular visio (VSD, VSDX, VSDM, VSSX, VSTX, VSTM, VDX, VTX, VST,VDW) file formats and allows exporting visio files to PDF, XPS, HTML, TIFF, JPG, PNG, BMP, and SVG。

最新的 API 参考可以在[Aspose.Diagram for .NET API 参考](https://reference.aspose.com/diagram/net).
