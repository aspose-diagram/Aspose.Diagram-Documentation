## **更新字段**
通过 Java 的 Python 的 Aspose.Diagram 允许您从您自己的应用程序中将字段更新为 Microsoft Visio 图表，无需 Microsoft Office 自动化。

Field 对象表示文本运行中的文本字段。 Shape 类公开的字段属性支持 Aspose.Diagram.Field 对象的集合。
